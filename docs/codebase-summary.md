# Codebase Summary

## Purpose

This repository is a reusable foundation for OpenCode-centered project setup.

## Current Shape

- `docs/` holds the operating model and standards
- `docs/opencode-agents/` holds OpenCode role specs
- `examples/opencode/` holds optional advanced config examples
- `.hive/` holds optional planning templates
- `pods/` holds pod-local memory
- `opencode.json` enables `opencode-hive`
