# Codebase Summary

## Purpose

This repository is a reusable foundation for OpenCode-centered project setup.

## Current Shape

- `docs/` holds the operating model and standards
- `docs/beads-standard.md` defines the default engineering execution model
- `docs/opencode-agents/` holds OpenCode role specs
- `examples/opencode/` holds optional advanced config examples
- `.hive/` holds optional planning templates and task handoff scaffolding
- `pods/` holds pod-local memory for the default `candidate-experience` and
  `content-cms-strapi` pods
- `scripts/` holds optional setup helpers such as Beads bootstrap
- `opencode.json` enables `opencode-hive`
