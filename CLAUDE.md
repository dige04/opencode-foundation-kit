# CLAUDE.md

This repository is a reusable OpenCode foundation starter.

Read first:
1. `docs/opencode-standard.md`
2. `docs/beads-standard.md`
3. `docs/agent-hive-standard.md`
4. `docs/integration-model.md`
5. `docs/code-standards.md`

Key rule: OpenCode supports implementation, but it does not become the system
of record.

This starter is Beads-first, with optional `.hive/` planning for teams that want
additional feature scaffolding.
